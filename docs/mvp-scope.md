# OPSBOARD — MVP Scope

## What MVP includes

1. **NOW** — operational attention screen
2. **INCIDENTS** — capture and manage operational exceptions
3. **OPERATIONS** — capture routine operational state
4. **MINIMAL ADMIN** — user/zone/service management
5. **OFFLINE-SAFE DRAFT CAPTURE** — local drafts with honest sync state

## What MVP explicitly excludes

- Trend analytics
- BI dashboards or charts in NOW
- Workflow builder
- Advanced alert automation engine
- Role explosion / complex permission builder
- Generic configurator platform
- Hardcoded operational numbers (staff, volunteer, zone counts)
- Destructive history rewrites
- Speculative v2 abstractions
- Bulk incident actions
- Deleting incidents
- Offline resolve / escalate / admin changes
- Silent conflict resolution or fake sync certainty
- CRDT or speculative local-first architecture
- Native mobile app
- HR/payroll/registration features
- KPI engine or historical comparisons
